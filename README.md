# Velora Market

React + TypeScript + Vite கொண்டு உருவாக்கப்பட்ட ஒரு modern e-commerce storefront starter.

## உள்ளவை

- responsive landing page
- featured product grid
- category filter chips
- add-to-cart interaction
- order summary side panel
- clean starter structure for future API/payment integration

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Customize செய்ய வேண்டிய இடங்கள்

- `src/data/products.ts` இல் real product data அல்லது API integration சேர்க்கலாம்
- `src/App.tsx` இல் auth, search, checkout flow, backend calls சேர்க்கலாம்
- `src/styles.css` இல் branding மற்றும் theme மாற்றலாம்
